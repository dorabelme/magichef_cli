
# Magichef 🥡

Simple Node CLI app built to manage ingredients, inventory and recipes.


## How to run locally? 💻

1. Clone and cd into the repo.
2. Run `npm install` to download dependencies.
3. Run `npm link`
4. Run `magichef` for instructions


## Tech Stack 🤖

- Node.js
- Commander
- Sqlite3
- Sequelize
- Colors
- Jest


## Testing 📍

To run tests, run `npm test`.
